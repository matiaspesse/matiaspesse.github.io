4015 ST / Einpunktstuhl, Bellmann-Stuhl

Contexto La silla de un punto, creada por Hans Bellman, fue una de las primeras sillas con una carcasa de asientos multiples curvas hecho de una unica pieza. Esta creacion fue una ocurrencia repentina de Bellman al ver el soporte de un portaequipajes.
